🌏PlanIt🌏
PlanIt is a modern and efficient web application for trip and itinerary planning. It allows users to easily organize attractions, activities, and schedules based on their preferences.

✨ Project Goals Allow users to plan multi-day trips.
Display attractions, restaurants, and points of interest.
Organize the trip by days and locations.
Show an interactive map with day-by-day markers.
Enable users to edit and customize their itinerary.
Keep the platform fast, responsive, and easy to use.
Build a strong foundation to add features like user login, favorite places, and budget management in the future.
